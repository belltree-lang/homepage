## 3. HPとLPの使い分け原則

1. **症状7ページが検索流入用LPを兼ねる**。すでに1ページ完結の説得構造（お悩み→帯①共感→できること→対象→帯③費用→相談CTA）を持っているので、症状別LPは別に作らない
2. **単目的LPは「流入を自分で制御できる経路」にだけ作る** → 当面は1本だけ：**無料体験LP**（チラシQR・SNSプロフィール・GBP投稿の共通受け皿）
   - 構成: 初回60分・国家資格者・保険のめやす（30分420円〜）・エリア5市・電話とフォームのみ。ナビなし
   - ブランド表記は「べるつりー（鈴木治療院）」併記でチラシとの橋渡し ※表記の最終形は制作時に代表確認
3. LPを乱立させない。増やすのは「計測して、足りないと分かってから」

## 4. SNS・GBPの役割定義

**最優先はGBP**（広告費ゼロで地域検索に直結。SNSより先に整える）:
- 鈴木治療院GBP: サイトリンクを訪問鍼灸詳細へ。写真更新・月2投稿・口コミ依頼の運用（体験後の家族に一声）
- べるフィットGBP: bellmfit.comへ。「保険が使えるジム」で統一
- 居宅事業所GBP: 居宅詳細ページへ

**SNSは「集客装置」ではなく「検索後の信頼確認装置」**と位置づける（更新停止が逆効果になるため、無理のない頻度で）:
- Facebook: 週1目安。事例・地域活動・教育型 → 症状ページへ
- Instagram: 週1目安。現場の瞬間（帯写真と同じ世界観）→ プロフィールリンク＝無料体験LP
- 投稿ネタは月次でまとめて作る（マーケ室で下書き→代表確認→予約投稿）
