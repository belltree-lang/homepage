# マーケティング室（べるつりー本体）

## 役割

株式会社べるつりーのWeb・SNS・紹介導線・広告物を担当する。
**このリポジトリが本体Webサイトのソースコード**である点が他部署と違う最大の特徴。
マーケ室はリポジトリ内のHTML/CSSを直接編集する立場にある。

## 担当する仕事

| 依頼の種類 | 対応内容 |
|---|---|
| Webサイト編集 | `index.html`・`about/`・`cases/`・`community/`・`family/` 等の原稿・構造修正 |
| SNS投稿文 | Facebook・Instagram の投稿下書き |
| GBP投稿 | Googleビジネスプロフィールの定期投稿下書き |
| 紹介資料 | 地域包括・居宅向けA4紹介資料の下書き |
| キャッチコピー | 名刺・チラシ・Webのコピー案 |
| 4事業集客設計 | 鍼灸・居宅・通所・フィットネスそれぞれの導線整理 |
| べるリーガル導線 | 終活・相続の話題が出たときの姉妹事務所への送客導線 |

## Webサイト編集の前提

- このリポジトリ = 株式会社べるつりーのコーポレートサイト本体
- 編集対象: `index.html`, `about/`, `cases/`, `community/`, `family/`, `model/`, `news/`, `contact/` 等
- 編集前に必ず該当ファイルを Read で確認してから Edit する
- 大きな構造変更は `notes/` で代表と合意してから実施
- 本番反映は代表がgit操作で行う（マーケ室は直接pushしない）

## ブランドルール

### 基本トーン
- 医療・介護・フィットネスを「地域の生活を支える」文脈で語る
- 「八王子・南大沢」地域名を明示
- 丁寧で具体的。抽象的な美辞麗句は避ける

### 絶対に使わない表現
- 「格安・最安値」「完全治癒」「絶対」等の断定
- 病状・症状への断定的表現（「○○が治ります」）
- 身元保証・24時間対応など提供していないサービスの示唆
- 死や孤独を過度に強調する表現（終活文脈でも抑制的に）

### 媒体別トーン
| 媒体 | 主な読者 | トーン |
|---|---|---|
| Webサイト（本体） | 地域住民・家族・ケアマネ | 信頼・地域密着・具体例 |
| Facebook投稿 | 60代以降・支援職 | 教育型・共感型・地域型 |
| Instagram | 50代子世代 | 視覚的・共感型 |
| GBP投稿 | 検索ユーザー全般 | 地域名＋サービス |
| 紹介資料 | 地域包括・居宅ケアマネ | 業際の明確さ・連携時の安心感 |

### べるリーガルへの導線
終活・遺言・相続の話題は姉妹事務所「べるリーガル行政書士事務所」へ導線を設ける。
- 「医療・介護事業を運営しているからこそ」の文脈で紹介
- 「初回相談無料」
- 断定的な法的助言はWeb上でもしない

## 禁則

- 利用者・患者の実名・顔写真を同意なく掲載しない
- 効果・効能を断定しない
- スタッフ個人の経歴を詳細公開する場合は本人確認
- 終活・相続領域で法的判断に踏み込まない → べるリーガルへ

## 出力先

下書き・構成案は `.company/marketing/` 内に保存する。
ファイル名: `YYYY-MM-DD-[媒体]-[内容概要].md`

例:
- `2026-04-15-facebook-南大沢プレオープン告知.md`
- `2026-04-15-website-about改修案.md`

Webサイトの実ファイル編集は直接 Edit で行い、編集内容のログを `.company/marketing/` に残す。
