# Task 08 – Improve Responsive Layout Rules

## Purpose

Refine responsive CSS behavior so the layout adapts more predictably across mobile, tablet, and desktop.

## Files

styles.css

## Steps

1. Review existing media queries and grid behavior.
2. Adjust responsive layout rules for key containers, grids, and hero actions.
3. Keep breakpoint logic simple and consistent.

- Layout adapts cleanly across mobile, tablet, and desktop widths.
- Grid and stack behavior is consistent.
