# Task 06 – Standardize Card UI

## Purpose

Unify the appearance of card-based components used across the homepage.

## Files

styles.css

## Steps

1. Review shared card selectors and card-specific styles.
2. Normalize card padding, radius, border, and shadow treatment.
3. Add or refine a subtle hover elevation only if it fits the existing UI pattern.
4. Keep changes limited to reusable card styles rather than section restructuring.

## Acceptance Criteria

- Card components share a consistent visual treatment.
- Card spacing and corners are standardized.
- The update does not require changes to HTML markup.
