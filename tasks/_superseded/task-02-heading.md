# Task 02 – Optimize Heading Structure

## Purpose

Improve the heading hierarchy in the homepage markup without changing business content.

## Files

index.html

## Steps

1. Review the heading levels used in the main content and footer.
2. Keep the main content heading order logical and sequential.
3. Replace non-structural footer headings with non-heading elements if they should not participate in the document outline.
4. Preserve the existing visible text unless a tag change is required.

## Acceptance Criteria

- The page has a clear heading hierarchy.
- Main content headings are logically ordered.
- Footer utility labels do not create unnecessary structural headings.
