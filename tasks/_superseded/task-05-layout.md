# Task 05 – Normalize Layout Width and Spacing

## Purpose

Bring the global layout width and spacing into a more consistent system.

## Files

styles.css

## Steps

1. Review container width, section padding, and common spacing tokens.
2. Adjust layout tokens to create consistent vertical rhythm across sections.
3. Keep spacing changes global and avoid section-specific redesign unless necessary.
4. Preserve the existing page structure.

## Acceptance Criteria

- The site uses a consistent container width.
- Section spacing feels uniform across the page.
- Changes are limited to layout and spacing rules in `styles.css`.
