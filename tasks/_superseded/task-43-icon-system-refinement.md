# Task 43 – Icon System Refinement

Goal

Refine the lightweight icon system so icons feel consistent across sections.

Scope

Mission, support model, and business unit icon wrappers and related CSS.

Files to Modify

index.html
styles.css
assets/icons

Changes

Adjust icon wrapper styling so icon blocks share one visual pattern.

Add optional icon variants only if they reuse existing tokens and reduce repetition.

Keep inline SVG usage lightweight and framework-free.

Constraints

Do not import icon libraries.

Do not add decorative images.

Use only neutral, token-based icon surfaces.

Acceptance Criteria

Icon wrappers use a consistent size, border, and background pattern.

Small icons remain visually aligned with standard icons.

SVG sizing and coloring remain predictable across sections.
