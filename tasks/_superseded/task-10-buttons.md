# Task 10 – Standardize Button Styles

## Purpose

Refine button styling so primary and secondary actions feel consistent across the site.

## Files

styles.css

## Steps

1. Review shared button styles and button variant rules.
2. Standardize spacing, colors, borders, and radius for button components.
3. Preserve visible focus states and restrained hover behavior.
4. Keep the change limited to button styles in `styles.css`.

## Acceptance Criteria

- Primary and secondary buttons use a consistent visual system.
- Focus and hover states remain visible and accessible.
- No HTML changes are required.
