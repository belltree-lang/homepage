# Task 09 – Improve Long-Form Readability

## Purpose

Make text-heavy sections easier to read without changing the content itself.

## Files

styles.css

## Steps

1. Review width and spacing of long-form text areas such as message and model sections.
2. Adjust line length, paragraph rhythm, and supporting spacing where necessary.
3. Keep changes focused on readability rather than visual redesign.
4. Avoid changing the HTML structure.

## Acceptance Criteria

- Long-form sections are easier to read.
- Text blocks do not become overly wide on large screens.
- Changes are limited to `styles.css`.
