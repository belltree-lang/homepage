# Task 50 – Visual Hierarchy System

Goal

Improve the consistency of headings, labels, key metrics, and supporting text across sections.

Scope

Section headers, lead text, metric treatment, and emphasis patterns.

Files to Modify

index.html
styles.css

Changes

Introduce reusable hierarchy helpers for labels, titles, intros, and highlighted values.

Clarify how primary and secondary text are visually distinguished.

Constraints

Do not change the document heading order.

Do not introduce one-off font sizes that bypass design-system.md.

Acceptance Criteria

Sections share a consistent visual hierarchy.

Key information is easier to scan without changing semantic headings.

Typography remains token-based and accessible.
