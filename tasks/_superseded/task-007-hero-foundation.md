# Task 007 – Hero Foundation

## Goal
ヒーローセクションの基本構成を整え、ページ冒頭の情報伝達を明確化する

## Scope
変更可能ファイル
- index.html
- styles.css

## Changes
具体的変更
- ヒーローの見出し、補助文、CTA の情報階層を整理する
- ヒーローの余白、配置、視線誘導を調整する
- 他セクションへ影響しない範囲でヒーロー専用スタイルを定義する

## Constraints
禁止事項
- framework追加禁止
- design token必須
- semantic HTML維持
- 既存の静的アーキテクチャを維持すること

## Acceptance Criteria
完了条件
- ヒーローが単体で企業価値を説明できる
- ファーストビューで主要CTAが認識できる
- CSS重複なし

## Dependencies
依存タスク
- Task 002 – Design Token Freeze

## Parallel Safety
衝突しない範囲
- ヒーロー領域に限定したHTML/CSS変更であれば並行作業可能
