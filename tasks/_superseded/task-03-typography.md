# Task 03 – Align Typography Scale

## Purpose

Adjust the typography rules to better match the intended hierarchy and readability of the corporate site.

## Files

styles.css

## Steps

1. Update heading font sizes to create clearer separation between `h1`, `h2`, and `h3`.
2. Refine body text size and line-height for comfortable reading.
3. Adjust supporting text styles such as labels and subtitles only where needed to preserve hierarchy.
4. Keep the font family and static-site approach unchanged.

## Acceptance Criteria

- Heading sizes form a clear visual hierarchy.
- Body text remains readable on desktop and mobile.
- Typography changes are limited to `styles.css`.
