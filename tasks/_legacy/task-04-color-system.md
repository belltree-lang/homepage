# Task 04 – Apply Approved Color System

## Purpose

Update the site color tokens so the visual palette is more consistent and controlled.

## Files

styles.css

## Steps

1. Review the current root color tokens.
2. Replace or normalize colors used for headings, buttons, backgrounds, borders, and accents.
3. Keep contrast sufficient for text and interactive elements.
4. Do not add new dependencies or change HTML structure.

## Acceptance Criteria

- Core colors are consistent across the site.
- Buttons, headings, surfaces, and backgrounds use a unified palette.
- Text and interactive elements remain readable.
