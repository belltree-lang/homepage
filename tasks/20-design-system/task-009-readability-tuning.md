# Task 009 – Readability Tuning

## Goal
本文可読性と情報の読み進めやすさを改善する

## Scope
変更可能ファイル
- styles.css
- index.html

## Changes
具体的変更
- 本文の行間、段落間隔、見出し間隔を見直す
- 長文セクションの最大行長と視認性を調整する
- 強調表現が多すぎる箇所を抑え、読み順を明確にする

## Constraints
禁止事項
- framework追加禁止
- design token必須
- semantic HTML維持
- 日本語本文の可読性を優先する

## Acceptance Criteria
完了条件
- 長文セクションを無理なく読める
- 見出しと本文の区別が明確である
- 読みやすさ改善のためのスタイルが重複していない

## Dependencies
依存タスク
- Task 002 – Design Token Freeze

## Parallel Safety
衝突しない範囲
- タイポグラフィ調整は全体影響が大きいため、全体CSS変更と同時進行しない
