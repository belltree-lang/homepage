# Task 019 – Localize Navigation Copy

## Goal
ナビゲーション文言をサイト全体の日本語トーンに合わせる

## Scope
変更可能ファイル
- index.html

## Changes
具体的変更
- ヘッダーとフッターのナビゲーションラベルを日本語化する
- セクション名とナビ文言の対応を揃える
- CTA系リンクと案内系リンクの言葉を整理する

## Constraints
禁止事項
- framework追加禁止
- design token必須
- semantic HTML維持
- 実際のセクション名と異なる文言にしない

## Acceptance Criteria
完了条件
- ナビゲーションが日本語として自然である
- ラベルと遷移先の内容が一致している
- 重複または曖昧な導線文言がない

## Dependencies
依存タスク
- Task 005 – Navigation Architecture

## Parallel Safety
衝突しない範囲
- ナビゲーション文言のみなら他の本文変更と並行可能
