# Task 031 – Homepage Hierarchy Copy Polish

## Goal
ホームページのヒーローと主要セクションの文言階層を整え、日本語UIとしての統一感とスキャン性を高める

## Scope
変更可能ファイル
- index.html

## Changes
具体的変更
- ヒーロー周辺の補助文とラベルを整理する
- 英語ラベルを日本語中心の表現へ統一する
- セクション導入文と見出しの関係を明確にする

## Constraints
禁止事項
- framework追加禁止
- design token必須
- semantic HTML維持
- 確定済みのヒーロー見出しは変更しない

## Acceptance Criteria
完了条件
- ヒーローの読み順が自然である
- セクションラベルと本文のトーンが統一されている
- UI崩れなし

## Dependencies
依存タスク
- task-009-cta-system-standardization

## Parallel Safety
衝突しない範囲
- ホームページ本文の文言と構造に限定すれば、共有CSS変更とは並行可能
