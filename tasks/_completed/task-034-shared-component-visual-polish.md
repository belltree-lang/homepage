# Task 034 – Shared Component Visual Polish

## Goal
共有コンポーネントの見た目を磨き、カード、ボタン、CTA、フッターの完成度を引き上げる

## Scope
変更可能ファイル
- styles.css

## Changes
具体的変更
- カード派生の見た目差を整理する
- ボタンとテキストCTAの視認性を改善する
- セクションヘッダー、CTA、フッターの視覚品質を高める

## Constraints
禁止事項
- framework追加禁止
- design token必須
- semantic HTML維持
- CSS重複を増やさない

## Acceptance Criteria
完了条件
- 共有コンポーネントの見た目に一貫性がある
- design-systemと一致
- CSS重複なし

## Dependencies
依存タスク
- task-008-card-component-variants
- task-009-cta-system-standardization
- task-010-trust-block-system

## Parallel Safety
衝突しない範囲
- 共有CSS変更のため、他のCSSタスクとは逐次実行が望ましい
