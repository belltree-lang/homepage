# Task 014 – Support Model Diagram

## Goal
支援モデルを図解し、文章だけでは伝わりにくい連続性を視覚化する

## Scope
変更可能ファイル
- index.html
- styles.css
- assets/images/*

## Changes
具体的変更
- 支援モデルの4段階を視覚的に接続する図またはレイアウトを導入する
- テキスト説明と図解の役割分担を整理する
- モバイルでも読める構造に調整する

## Constraints
禁止事項
- framework追加禁止
- design token必須
- semantic HTML維持
- 画像だけに依存して内容を伝えない

## Acceptance Criteria
完了条件
- 支援の連続性が視覚的に理解できる
- 画像がなくても内容の理解が成立する
- モバイル表示で図解が破綻しない

## Dependencies
依存タスク
- Task 008 – Responsive Layout

## Parallel Safety
衝突しない範囲
- 支援モデルセクションに限定すれば他セクション作業と並行可能
