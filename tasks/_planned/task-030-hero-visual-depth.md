# Task 030 – Hero Visual Depth

## Goal
ヒーローに立体感と奥行きを加え、静的サイトでも印象の深いファーストビューをつくる

## Scope
変更可能ファイル
- index.html
- styles.css
- assets/images/*

## Changes
具体的変更
- 背景レイヤー、装飾レイヤー、前景コンテンツの関係を整理する
- 影、ぼかし、グラデーション、図形の使い方を洗練する
- 過剰な演出にならない範囲で深みを追加する

## Constraints
禁止事項
- framework追加禁止
- design token必須
- semantic HTML維持
- 可読性を損なう重い演出を入れない

## Acceptance Criteria
完了条件
- ヒーローに奥行きが感じられる
- コンテンツの可読性とCTA視認性が維持されている
- モバイルでも演出が破綻しない

## Dependencies
依存タスク
- Task 020 – Hero Visual Balance

## Parallel Safety
衝突しない範囲
- ヒーロー専用装飾に限定すれば並行可能
