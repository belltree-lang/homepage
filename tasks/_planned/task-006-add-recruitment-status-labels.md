# Task 006: Integrate Recruitment Status Labels

## Title
Show active recruitment status and roles on the recruitment top page.

## Problem
The recruitment page lacks live-status indicators, forcing candidates to read deeply to know if their role is currently hiring.

## Target Files
- `pages/recruit/index.html`

## Implementation

- Implement a clear list of active openings at the top of the recruitment page.

## Acceptance Criteria
- [ ] Hiring status is visible at a glance on the recruitment top page.
