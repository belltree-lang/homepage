# Task 021 – Section Contrast

## Goal
セクションごとの視覚コントラストを調整し、ページ全体のリズムを整える

## Scope
変更可能ファイル
- styles.css
- index.html

## Changes
具体的変更
- 背景、境界線、カード面の対比を再調整する
- 同系色が続きすぎるセクションに変化を持たせる
- 情報の重要度に応じたコントラスト差を設計する

## Constraints
禁止事項
- framework追加禁止
- design token必須
- semantic HTML維持
- 単純に色数を増やして解決しない

## Acceptance Criteria
完了条件
- ページを通して単調に見えない
- コントラスト調整後もブランドトーンを維持している
- セクション差分がCSSで一貫して管理されている

## Dependencies
依存タスク
- Task 012 – Section Background Contrast

## Parallel Safety
衝突しない範囲
- 全体配色に関わるため、他の全体スタイル変更とは競合しやすい
