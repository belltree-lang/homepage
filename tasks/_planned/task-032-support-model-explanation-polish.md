# Task 032 – Support Model Explanation Polish

## Goal
支援モデルの説明を、文章と構造の両面から分かりやすく整える

## Scope
変更可能ファイル
- index.html

## Changes
具体的変更
- 支援モデルのステップ説明を補強する
- 各ステップの役割差を明確にする
- セクション単体で意味が通るように情報順を調整する

## Constraints
禁止事項
- framework追加禁止
- design token必須
- semantic HTML維持

## Acceptance Criteria
完了条件
- 支援モデルの流れが読みやすい
- ステップ間の違いが明確である
- UI崩れなし

## Dependencies
依存タスク
- task-010-trust-block-system

## Parallel Safety
衝突しない範囲
- 支援モデルセクションに限定すれば他セクションHTML変更と並行可能
