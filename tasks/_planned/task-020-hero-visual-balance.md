# Task 020 – Hero Visual Balance

## Goal
ヒーロー内の文字量、余白、装飾量のバランスを調整する

## Scope
変更可能ファイル
- index.html
- styles.css

## Changes
具体的変更
- 見出し、補助文、CTA の比重を再調整する
- 装飾要素と本文の距離感を整える
- モバイルとデスクトップで密度差が出すぎないよう補正する

## Constraints
禁止事項
- framework追加禁止
- design token必須
- semantic HTML維持
- 文言確定済みのコピーは変更しない

## Acceptance Criteria
完了条件
- ヒーローの主役が見出しであることが明確
- 情報量が多く見えすぎない
- どの画面幅でもCTAが埋もれない

## Dependencies
依存タスク
- Task 011 – Hero Visual Design

## Parallel Safety
衝突しない範囲
- ヒーロー内のスタイルに限定すれば並行可能
