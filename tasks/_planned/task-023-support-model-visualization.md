# Task 023 – Support Model Visualization

## Goal
支援モデル図解を実装レベルで洗練し、理解しやすい情報コンポーネントに仕上げる

## Scope
変更可能ファイル
- index.html
- styles.css
- assets/images/*

## Changes
具体的変更
- 図解の情報順、矢印、接続表現、注釈の見せ方を改善する
- カード型やステップ型など、最適な表現へ整える
- 視覚補助と本文の連携を強める

## Constraints
禁止事項
- framework追加禁止
- design token必須
- semantic HTML維持
- 装飾過多で内容理解を阻害しない

## Acceptance Criteria
完了条件
- 支援モデルの流れが一目で追える
- 説明文と図解が相互補完している
- レスポンシブでも図解の意味が壊れない

## Dependencies
依存タスク
- Task 014 – Support Model Diagram

## Parallel Safety
衝突しない範囲
- 支援モデルセクションに限定すれば並行可能
