# Task 022 – Icon System

## Goal
定義済みアイコン方針を実装へ反映し、UIに統一感を持たせる

## Scope
変更可能ファイル
- assets/icons/*
- index.html
- styles.css

## Changes
具体的変更
- 必要なセクションへアイコンを適用する
- テキストとアイコンの整列、余白、サイズを調整する
- アセット追加時の命名規則と使用箇所を揃える

## Constraints
禁止事項
- framework追加禁止
- design token必須
- semantic HTML維持
- アイコンだけに情報伝達を依存させない

## Acceptance Criteria
完了条件
- アイコン適用箇所に一貫性がある
- アセット命名と利用ルールが揃っている
- アイコン未読でも本文理解に支障がない

## Dependencies
依存タスク
- Task 013 – Icon System Foundation

## Parallel Safety
衝突しない範囲
- アイコン導入対象セクションごとに分ければ一部並行可能
