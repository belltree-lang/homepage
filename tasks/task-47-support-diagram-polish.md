# Task 47 – Support Diagram Polish

Goal

Improve the structure and readability of the support model diagram.

Scope

Support model diagram wrapper, node layout, and supporting labels.

Files to Modify

index.html
styles.css

Changes

Refine the support diagram so each node has a clearer internal hierarchy.

Add lightweight labels or helper text only if they support readability.

Keep the diagram grid-based and static.

Constraints

Do not replace the diagram with an image.

Use existing design tokens for borders, surfaces, spacing, and typography.

Acceptance Criteria

Support nodes have a clearer label hierarchy.

The diagram remains responsive and readable on narrow screens.

The visual treatment stays consistent with the card system.
