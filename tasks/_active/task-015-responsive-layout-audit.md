# Task 015 – Responsive Layout Audit

## Goal
レスポンシブ表示の崩れや不整合を洗い出し、調整方針を明確にする

## Scope
変更可能ファイル
- index.html
- styles.css

## Changes
具体的変更
- モバイル、タブレット、デスクトップで主要セクションを点検する
- 余白、折り返し、列数、CTA配置の不整合を洗い出す
- 必要な修正項目と優先順位を整理する

## Constraints
禁止事項
- framework追加禁止
- design token必須
- semantic HTML維持

## Acceptance Criteria
完了条件
- 主要なレスポンシブ課題が把握できている
- design-systemと一致
- 画面幅ごとの改善方針が明確である

## Dependencies
依存タスク
- task-014-grid-system-cleanup

## Parallel Safety
衝突しない範囲
- 監査中心の作業であれば他実装作業と並行可能
