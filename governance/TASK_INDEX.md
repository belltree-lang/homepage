# Task Index

## Purpose

Define stable task number ranges for the repository.

## Ranges

- `001-006` Foundation governance and task operating system
- `007-015` Initial page shell, component, CTA, trust, and layout stabilization
- `016-030` Site refinement, accessibility, SEO, and structural cleanup
- `031-049` Future page expansion and content modules
- `050-069` Content, localization, SEO, and accessibility follow-up work
- `070-089` Assets, QA, and validation work
- `090-099` Review, refactor, and migration work

## Notes

- New task numbers should be assigned from the appropriate range.
- Superseded legacy tasks keep their original filenames in `_superseded`.
