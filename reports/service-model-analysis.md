# BellTree Service Model Analysis

本レポートは、`reports/final-site-analysis.md` で特定された課題をさらに深掘りし、BellTreeの独自の統合型地域支援モデル（Integrated Model）が、ウェブサイトを通じてどのようにユーザーに伝わり、行動（コンバージョン）を促しているかをビジネス視点で分析したものです。

---

## 1. サービスモデルの伝達状況 (Service Communication Gaps)

BellTreeの最大の特徴である「人生のステージに合わせたシームレスな支援（フィットネス → 訪問 → 介護 → 法務）」は、図解（ecosystem-flow）やステップ表記によって論理的には提示されていますが、以下の点において伝達のギャップが存在します。

- **「変化の予兆」へのメッセージ:** 各サービスは「今困っていること」への対応（点）としては優秀ですが、フィットネスから訪問、あるいは訪問から介護へ「いつ、どのようなきっかけで移行するのか」という動的なつながり（線）の描写が不足しています。
- **コミュニティ支援の統合不足:** `pages/services/community/ index.html` で紹介されている素晴らしい地域活動（SOFTエクササイズ等）が、他の専門サービスとどのようにリンクしているかが不明確です。「活動に参加することが、将来の安心（相談導線）にどう繋がるか」という、社会的処方の核となる部分の言語化が弱いです。
- **専門職間の連携エピソード:** 「国家資格者のチーム」という言葉はありますが、実際にケアマネジャーと理学療法士、あるいは行政書士がどのように「一人の利用者」の情報を共有し、最善の策を講じているかという「連携の現場感」が伝わっていません。

## 2. ユーザー状況別の入り口分析 (User Scenario Coverage)

サイトが想定される主要な5つのエントリーシナリオをどの程度カバーしているかを評価します。

- **将来の健康が不安なシニア [評価: A]**
  - 「べるフィット」の導線が明確であり、予防への入り口として機能しています。
- **急な介護に直面したご家族 [評価: B-]**
  - 居宅介護支援のページはありますが、急な事態でパニックになっている家族に対する「まずこれを読んでください」というクイックガイドや、緊急対応の安心感が不足しています。
- **慢性的な痛みを抱える方 [評価: B+]**
  - 訪問鍼灸マッサージのターゲットは明確ですが、医療保険適用などの「制度的な安心感」と「技術的な信頼感」のバランスにおいて、技術面（治るイメージ）の訴求がややテキスト主体です。
- **採用候補者 [評価: A-]**
  - 採用トップページの情報量は非常に豊富で、特に「数字で見る」「離職率0%」は強力なフックになっています。ただし、職種ごとの「一日中どのような動きをするか」というリアルな時間軸の乖離があります。
- **地域パートナー（医療機関・企業） [評価: C]**
  - 連携の実績（バラエティクラブ等）は掲載されていますが、新たなパートナーが「BellTreeと一緒に何を創れるか」をイメージさせるためのB2B向けメッセージや、提携のメリットが整理されていません。

## 3. 信頼の設計図 (Trust Architecture Review)

現在のウェブサイトは「客観的信頼（数字、資格）」は非常に高いですが、「主観的信頼（安心感、人柄）」の構築に課題があります。

- **医療的信頼:** 資格名や沿革、実績回数で一定のレベルを担保。
- **地域的信頼:** 町田市・南大沢エリアに根ざした活動実績と具体的な提携先名で十分に構築。
- **スタッフの可視性 [最重要課題]:** 採用ページにはスタッフ紹介がありますが、サービス提供ページには「実際に誰が来るか」の顔が見えません。医療・介護において「人」は最大のサービス内容そのものです。
- **利用者・ケース例 [不足]:** 「町田市在住70代」といったモデルケースはありますが、実名や写真（許可の範囲内）を伴う「本物の声」がないため、実感が湧きにくい状態です。

## 4. コンバージョン導線の弱点 (Conversion Path Weaknesses)

ユーザーが「情報収集（Info）」から「信頼（Trust）」を経て「相談（Consultation）」に至るまでのボトルネックを特定します。

- **情報過多による思考停止:** 特に採用ページやサービス一覧において、文章量が多く、ユーザーが「自分に関係があるのはどこか」を判断する前に疲弊する可能性があります。
- **「相談」の具体性不足:** 「ご相談窓口」をクリックした後に何が起きるのか（電話で何を聞かれるのか、訪問されるのか等）の不安が、最後の一歩を躊躇させています。
- **検討度合いに応じた導線の欠如:** 「今すぐ来てほしい」人と「将来のためにパンフレットがほしい」人が同じ窓口に誘導されており、ライトなコンバージョン（資料請求やLINE登録等）が用意されていません。

## 5. 戦略的なコンテンツ追加案 (Strategic Content Additions)

信頼とコンバージョンを飛躍させるための高インパクトな追加施策を提案します。

- **1. 連携のドキュメンタリー「統合支援の24時間」:** 一人の利用者の生活を、トレーナー、施術者、ケアマネがバトンを繋ぐように支える様子を追ったドキュメント（またはインフォグラフィック）。
- **2. スタッフ・コンシェルジュ紹介:** 各サービス領域のリーダーが「なぜこの仕事をしているか」を語るインタビュー動画または記事。「第二の家族」を体現する人柄の可視化。
- **3. 「地域包括ケアの教科書」資料提供:** 地域の介護・医療制度を分かりやすくまとめたBellTree謹製のPDFガイド。「相談」の前に「教育的価値」を提供することで圧倒的な信頼を構築。
- **4. オンライン見学会/体験レポート:** 「べるフィット」のトレーニング風景や訪問施術のデモンストレーションをVRや動画で疑似体験できるコンテンツ。

## 6. 優先改善事項 (Priority Improvements)

1.  **各サービスページへの「担当スタッフの顔」掲載:** 専門資格だけでなく、信頼できる人柄を視覚的に提示。
2.  **Web問い合わせフォームの多層化:** 「急ぎの相談」「見学・体験予約」「資料請求」を分けたフォーム設置。
3.  **B2B/地域共創ページの独立:** 地域包括支援センターや他企業に向けた「連携のメリットと窓口」を明確化。
4.  **社会的処方の言語化:** ニュースや地域活動セクションにおいて、「活動への参加がどのように将来の安心（予防）に直結するか」を定義し、全ページで一貫して伝える。
