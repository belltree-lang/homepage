BellTree Model

支援を途切れさせない、
人生後半の伴走者として。

私たちは、運動から介護、終活までを一つの流れとしてつなぐことで、地域の人々が長く安心して相談できる「つながり」そのものを作っています。

人生の「点」ではなく「線」を支える

べるつりーが目指すのは、心身の健康から介護、そして人生の締めくくりまでを一貫して支える「地域包括ケア」の新しい形です。

ライフステージに応じたシームレスな支援

1. 予防・フィットネス

元気なうちから始める運動習慣。姿勢改善や体幹強化で、要介護状態にならない身体作りを支援します。

2. 医療・リハビリ

痛みや不調が出た際の迅速な対応。訪問鍼灸やマッサージで、在宅での機能回復をサポートします。

3. 介護・生活支援

ケアマネジャーが中心となり、最適な制度活用を提案。ご本人とご家族が安心して暮らせる環境を整えます。

4. 法務・終活準備

財産管理や意志決定のサポート。最期まで自分らしく生きるための法的根拠を専門家が形にします。

私たちが大切にする3つの約束

🤝

一貫性

担当が変わっても、あなたの歴史や想いはチーム全員で共有されます。

🚀

即応性

地域のネットワークを活かし、困った時に「すぐ動ける」距離感を大切にします。

💡

専門性

各分野のプロが集まり、医学的・法的な根拠に基づいた支援をご提供します。
