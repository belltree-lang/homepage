スタッフ紹介

高い専門性と、一人ひとりに寄り添う心を持ったプロフェッショナル集団です。

👤
Message from Representative

鈴木 俊平

代表取締役 / 行政書士

運動・医療・介護、そして終活の分野が分断されている現状を変えたい。そんな想いから「べるつりー」は始まりました。私たちが提供したいのは「点」のサービスではなく、状態が変わっても途切れることのない「一本の線の支援」です。地域の皆様にとっての「第二の家族」のような存在として、専門性と温かさを両立し、人生の後半戦を共にあゆむパートナーであり続けたいと考えています。

多職種の専門家が、あなたの生活をチームで支えます

👤

丸山 裕久

鍼灸師 / 町田・多摩エリア院長

単に「痛みを取る」ことだけでなく、患者様が本当に大切にしている「やりたいこと」に寄り添う施術を心がけています。

👤

柳内 克尚

鍼灸マッサージ師 / 町田・多摩エリア副院長

接客業などの経験も活かし、「一人ひとりの心に寄り添う」ことを大切にしています。不調や不安を和らげ、日々の生活が前向きになるよう、誠実な施術と丁寧な対話を重ねて参ります。

👤

菅原 直志

鍼灸師 / 八王子・日野エリア副院長

痛みの改善だけでなく、その原因となる身体の使い方や筋力バランスにも注目し、施術と運動指導を組み合わせたサポートを行っています。

👤

中澤 仁

鍼灸師 / 八王子・日野エリア副院長

西洋医学と東洋医学の双方の視点から身体の状態を捉え、生活に寄り添った施術を提供しています。

👤

黒飛 勇人

健康運動指導士 / べるフィット施設長

運動を継続できる環境づくりを大切にし、利用者の皆さまが楽しく身体を動かせるようサポートしています。

👤

山本 誠一

べるフィット副施設長

会員様が「ここに来ると自然と元気が出る」と感じていただける場所づくりを心がけています。「今の一歩先」へ踏み出す皆様を一番近くで伴走し応援します。

👤

熊倉 篤史

主任介護支援専門員 / べるつりー居宅主任ケアマネジャー

ご本人やご家族の想いを整理しながら、制度や地域資源をつなぐケアマネジメントを行っています。

👤

三好 裕美

べるフィットマネージャー

べるフィットの運営マネージャーとして、運動習慣を無理なく続けられる環境づくりを大切にしています。健康づくりは運動だけでなく、人との関係や日常の楽しみの中で育まれるものだと考えています。

👤

松下 早苗

ストレッチトレーナー / べるフィットスタッフ

ストレッチや軽運動のサポートを担当しています。身体を動かすことが苦手な方でも安心して参加できるよう、一人ひとりのペースに合わせた丁寧なサポートを心がけています。

私たちが全力でサポートします
