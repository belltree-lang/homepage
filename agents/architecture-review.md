# Architecture Review Agent

## Purpose

Analyze the repository structure and detect architectural or design issues.

## Scope

`index.html`  
`styles.css`  
`design-system.md`  
`ARCHITECTURE.md`  
`tasks/`

## Responsibilities

Detect:

- HTML structure problems
- CSS duplication
- design-system drift
- unused selectors
- inconsistent spacing rules
- accessibility problems
- responsive inconsistencies

## Output

Create a report in:

`reports/architecture-review-{date}.md`

## Report structure

Repository overview  
HTML structure review  
CSS architecture review  
Design system compliance  
Architecture governance consistency  
Suggested improvements
