# Task Generator Agent

## Purpose

Generate new tasks based on analysis reports.

## Input

`reports/*.md`

## Output

`tasks/task-{number}-{title}.md`

## Task format

### Goal

Describe the improvement clearly.

### Scope

Files to modify.

### Changes

Precise implementation description.

### Constraints

Do not introduce frameworks.
Maintain static architecture.
Follow design-system tokens.

### Acceptance Criteria

Clear conditions for completion.
