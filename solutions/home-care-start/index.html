Solutions: Home Care Start

介護の始め方がわからない・家族の負担

親の様子が少し変わった、でも何から始めればいい？ 介護の「入り口」で立ち止まっているご家族の不安に、専門家が寄り添います。

このようなことはありませんか？

  • ✓
    最近、親が同じことを何度も聞くようになったり、忘れ物が増えた
  • ✓
    実家の掃除が行き届いていない、冷蔵庫の管理ができていない
  • ✓
    介護保険の申請をしたいが、手続きが複雑でよくわからない
  • ✓
    仕事と介護の両立に限界を感じ始めている

「まだ早い」と思っている間に。

介護は、ある日突然始まります。しかし、本格的なトラブルが起きてから動くのでは、ご本人もご家族も疲弊してしまいます。「少し気になる」という今の段階から、制度の仕組みを知り、外部のサポートを検討しておくことが、心の余裕に繋がります。

べるつりーの役割

私たちは、行政の窓口や病院のソーシャルワーカーとはまた違う立場から、お一人おひとりの「暮らし」に密着したアドバイスを行います。まだ介護認定を受けていない方の相談も、広く承っております。

べるつりーの統合サポート

点在する課題を整理し、チームで「暮らしの再設計」をお手伝いします。

📝

ケアマネジメント
(制度活用)

複雑な介護保険制度の活用を分かりやすく解説。最適なケアプランを作成し、各サービスとの調整を一手に引き受けます。

🏠

在宅ケアの実施
(訪問施術)

介護を導入する前段階として、訪問マッサージ等で専門家が定例的に訪問。ご本人の変化をいち早く察知する目となり、安心を届けます。

⚖️

将来の法務調整
(べるリーガル)

財産管理や成年後見など、介護が進むにつれて生じる法的な懸念を、専門の行政書士が早期にクリアにします。

解決のイメージ

遠方にお住まいのご家族の事例

離れて暮らす親の物忘れが心配でご相談。弊社の「訪問マッサージ」を導入することで、週2回の安否確認が可能に。その後、状態に合わせて社内の「ケアマネジャー」と連携し、介護認定の申請からデイサービスの利用開始までスムーズに進行。仕事との両立も続けられています。
